Website Creation and Deployment Process:

  1. Use Heavens Tools to create your website. Once you are satisfied with your site, publish it to a custom subdomain of heavens.pictures created through the Heavens Tools website creation wizard.
  2. Use the tool provided in this page to input the subdomain you created and provide a title for your website. This will generate a single index.html file specifically tailored to your website.
  3. Upload the generated index.html file to a hosting platform:
    • :
      1. Create an account on Fleek and connect your GitHub account.
      2. Select "From GitHub" and choose the repository for your website.
      3. Select IPFS as the hosting service.
      4. Fleek will deploy your website to IPFS and generate an IPNS key.
    • :
      1. Install IPNS Desktop and make sure IPFS is running.
      2. Drag and drop the generated index.html file into IPNS Desktop.
      3. IPNS Desktop will automatically generate an IPNS key for your website.
    • :
      1. Create an account on Pinata and log in.
      2. Upload your generated index.html file.
      3. Pinata will generate an IPFS hash for your file.
  4. Copy the generated IPNS key or IPFS hash.
  5. Go to the ENS Manager App and access the control panel for your .eth domain.
  6. In the Content field for your ENS name, paste the IPNS key or IPFS hash.
  7. Save the record and confirm the transaction in your wallet.
  8. Congratulations! Your website is now accessible via a decentralized domain.
