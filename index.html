Website Creation and Deployment Process:

  1. Use Heavens Tools to create your website. This is a drag-and-drop editor, making it simple to create a website without any coding knowledge.
  2. Download your website as a compressed .zip file once you're satisfied with your creation.
  3. Extract the compressed archive on your computer.
  4. Create a new repository on GitHub and upload your website files there.
  5. Sign up for an account on Fleek and connect it with your GitHub account. Add a new site and choose the repository you created earlier. Make sure IPFS is selected as the hosting service.
  6. Fleek will deploy your website to IPFS. When completed, copy the IPFS CID that is displayed.
  7. Go to the ENS Manager App and add the IPFS CID to the Content field for your ENS name. Save the record and confirm the transaction in your wallet.
  8. Congratulations! Your website is now accessible via a decentralized domain.
